数学建模 - 优化问题

所谓最优化问题是指使得问题中某一项指标“最优”的方案

优化问题建模：

  • 找到目标函数，也就是要找到“最优”
  • 寻找决策变量，也就是可以优化的变量
  • 确定约束条件，并用关于决策变量的数学语言表示
  • 根据变量的物理性质研究变量是否有非负性

优化问题求解：

  • 凸优化问题（目标函数为凸函数且可导）：梯度下降法、牛顿迭代法
  • 非凸优化问题：启发式优化算法（智能优化算法）
    • 粒子群算法：决策变量为连续变量
    • 遗传算法：决策变量为离散变量
    • 模拟退火算法，决策类型无要求，但维度较低

模型转换

首先要将求解问题的模型转化为 Linprog 的标准形式：
（1）求最大值问题要转换为求最小值问题：C = [-2, 3, 5]
（2）当约束条件为 大于等于 时要加负号：A_ub = [[-2, 5, -1], [1, 3, 1]]
（3）由 x1, x2, x3 >= 0 和 x1 + x2 + x3 = 7 可知：0 <= x1, x2, x3 <= 7

2. PuLP 库

https://www.cnblogs.com/youcans/p/14714085.html

PuLP是一个开源的第三方工具包，可以求解线性规划、整数规划、混合整数规划问题。

求解步骤

（0）导入 PuLP库函数
（1）定义一个规划问题
  • pulp.LpProblem 是定义问题的构造函数。
  • “LPProbDemo1”是用户定义的问题名（用于输出信息）。
  • 参数 sense 用来指定求最小值 / 最大值问题，可选参数值：LpMinimize、LpMaximize
（2）定义决策变量
  • pulp.LpVariable 是定义决策变量的函数
  • ‘x1’ 是用户定义的变量名
  • 参数 lowBound、upBound 用来设定决策变量的下界、上界；可以不定义下界/上界，默认的下界/上界是负无穷/正无穷。
  • 本例中 x1,x2,x3 的取值区间为 [0,7]
  • 参数 cat 用来设定变量类型，可选参数值：’Continuous’ 表示连续变量（默认值）、’ Integer ‘ 表示离散变量（用于整数规划问题）、’ Binary ‘ 表示0/1变量（用于0/1规划问题）。
（3）添加目标函数

添加目标函数使用 “问题名 += 目标函数式” 格式。

（4）添加约束条件
  • 添加约束条件使用 “问题名 += 约束条件表达式” 格式。
  • 约束条件可以是等式约束或不等式约束，不等式约束可以是 小于等于 或 大于等于，分别使用关键字”>=”、”<=”和”==”。
（5）求解
  • solve() 是求解函数。PuLP默认采用 CBC 求解器来求解优化问题，也可以调用其它的优化器来求解，如：GLPK，COIN CLP/CBC，CPLEX，和GUROBI，但需要另外安装。
